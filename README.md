# citySelect
仿美团等城市选择，自定义数据库

Step 1.Add it in your root build.gradle at the end of repositories:
```
allprojects {
		repositories {
			...
			maven { url 'https://www.jitpack.io' }
		}
	}
  ```
  Step 2. Add the dependency
 ``` 
  dependencies {
	        implementation 'com.github.fg2q1q3q:citySelect:1.0'
	}
  ```
